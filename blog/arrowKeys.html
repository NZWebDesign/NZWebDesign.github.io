Exploring some JavaScript...

Change the photos...

I wanted to experiment with some of the JavaScript from the carnival so I decided to create a page that would allow the user to select from three different type of photos, - Planes, Scenery, and Seagulls! The UP and DOWN arrow keys select from this main menu, and the LEFT and RIGHT arrow keys change the photo.

Have a try changing the "change the photo" box below with the arrow keys!

REMEMBER --- UP and DOWN arrow to select type of photo. LEFT and RIGHT to cycle through all the photos in that type....

Stop the page scrolling with the arrow keys!!...

One thing I discovered was that while using the up and down arrow keys, the page still scrolled up and down, and this was annoying so I needed to stop that. A quick google search turned up some code at Stack Overflow. It seems to have done the trick!! ... see the code below...

Photo editing...

I found Pixillion Image Converter was useful to batch convert photos to have a longest size of 700 pixels. The photos in their original size would slow page loading times, so needed to be resized. Pixillion Image Converter was a free download, and was easy to use, provided a good UX for this task with batch conversion done in only a few clicks.

I also discovered you can rename multiple files, which was useful as I wanted the files to have the same name apart from the index number. In the file explorer folder, if you click on a file and up the top select rename - name your file, and hit 'tab' and all the files will have the same name apart from a different number .
